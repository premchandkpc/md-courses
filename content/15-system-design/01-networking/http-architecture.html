🔒 TLS 1.3

1-RTT handshake (vs 2-RTT for TLS 1.2). 0-RTT for resuming. Perfect forward secrecy via ephemeral Diffie-Hellman. AEAD ciphers (AES-GCM, ChaCha20-Poly1305).

📨 HTTP/1.1

Text-based protocol. Pipelining (rarely used due to HOL blocking). Connection: keep-alive for reuse. Host header for virtual hosting.

🌀 HTTP/2

Binary framing layer. Multiplexed streams over single TCP connection. Server push. Header compression (HPACK). Stream prioritization.

🚀 HTTP/3

QUIC over UDP. 0-RTT handshake. No TCP HOL blocking. Connection migration (survives IP change). Mandatory encryption.
