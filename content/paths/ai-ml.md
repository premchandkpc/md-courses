# AI / ML Engineer

Build AI engineering expertise — from fundamentals through deep learning, LLMs, and production ML systems.

## Prerequisites

- [Backend — Junior](backend-junior.md) path or equivalent
- [Data Engineer](data.md) path recommended

## Path

1. **ML Fundamentals**
   - [Machine Learning Basics](/01-ai-ml/01-machine-learning-basics.md) — 60m

   - [Model Evaluation](/01-ai-ml/03-model-evaluation.md) — 30m

   - [Prompt Engineering](/01-ai-ml/05-llm/02-prompt-engineering.md) — 30m

   - [Feature Stores](/02-data-engineering/12-feature-stores.md) — 30m

6. **Production AI**
   - [A/B Testing for ML](/01-ai-ml/08-production/01-ab-testing.md) — 30m
   - [ML System Design](/01-ai-ml/08-production/02-ml-system-design.md) — 45m
   - [AI Agents](/01-ai-ml/09-agents/01-ai-agents.md) — 45m

## Outcome

You can design, build, and deploy ML systems in production. You understand the full stack from model training through deployment, monitoring, and iteration.
