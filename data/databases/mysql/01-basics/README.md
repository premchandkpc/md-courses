# MySQL Basics

MySQL is the world's most popular open-source relational database management system (RDBMS).

## Topics Covered

- **Database & Table Creation** - DDL, schemas, data types
- **Data Types & Constraints** - VARCHAR, INT, DECIMAL, PRIMARY KEY, FOREIGN KEY
- **Basic Queries** - SELECT, WHERE, INSERT, UPDATE, DELETE
- **Indexes** - B-tree, UNIQUE, PRIMARY KEY indexes
- **Joins** - INNER, LEFT, RIGHT, FULL OUTER joins

## Key Characteristics

- **Relational**: Structured schema with tables and relationships
- **ACID**: Transactions (with InnoDB storage engine)
- **Open Source**: Free, widely supported
- **Performance**: Optimized for read-heavy workloads
- **Scaling**: Vertical scaling, replication for reads

## Learning Path

1. Start with table design and data types
2. Master SELECT queries and WHERE clauses
3. Learn indexes for query optimization
4. Understand joins for multi-table queries
5. Progress to intermediate topics (subqueries, CTEs)
