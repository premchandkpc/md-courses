Active Component: Tabs
Composition: Flexible
Compound components split behavior across sub-components that share context. No 50-prop monolith - just composable pieces.
Context flow: Parent component owns state, subcomponents consume it via useContext hook.
Benefits: Flexible layouts, smaller props API, cleaner JSX, powers Headless UI & Radix.
