# Oracle Database Basics

Oracle Database is an enterprise-grade relational database management system known for scalability, security, and advanced features.

## Topics Covered

- **Database & Tablespace Creation** - Storage structures, datafiles
- **Data Types & Constraints** - VARCHAR2, NUMBER, DATE, constraints
- **Basic Queries** - SELECT, WHERE, INSERT, UPDATE, DELETE
- **Indexes** - B-tree, bitmap, function-based indexes
- **Joins** - All join types, SQL vs ANSI syntax
- **Sequences & Triggers** - Auto-increment, automatic actions

## Key Characteristics

- **Enterprise Grade**: Built for mission-critical systems
- **Scalability**: Handles massive databases (terabytes+)
- **Security**: Row-level security, encryption, auditing
- **Performance**: Advanced query optimizer, caching
- **Advanced Features**: Partitioning, compression, parallel query

## Learning Path

1. Understand tablespaces and storage structures
2. Master table creation with constraints
3. Learn indexing strategies
4. Understand joins and complex queries
5. Progress to advanced features (partitioning, compression)
