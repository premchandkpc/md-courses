🕐 Network Latency

DNS resolution (1-50ms), TCP handshake (1-3 RTT), TLS negotiation (2-3 RTT), geographic distance (speed of light ~1ms per 100km)

💾 Cache Latency

L1: ~1ns, L2: ~10ns, RAM: ~100ns, SSD: ~100µs, Network: ~1ms. Cache hierarchy reduces P99 dramatically.

🗄️ Database Latency

Index seek vs full scan, read replicas for P99, connection pooling, prepared statement caching, query optimization.

📊 Tail Latency

P99/P999 optimization: hedging requests, request coalescing, GC tuning, load shedding at saturation.

P99 Target< 200ms for web APIs, < 50ms for real-time
MeasurementDistributed tracing (Jaeger/Zipkin), metrics (Prometheus), RUM (Real User Monitoring)
StrategiesCDN edge caching, connection pooling, read replicas, HTTP/2 multiplexing, QUIC 0-RTT
Anti-patternsN+1 queries, thundering herd, hot keys, chatty APIs, unbounded in-memory caches
