Memory Management Architecture
Virtual Memory · Paging · TLB · Allocation
Virtual memory lets 16GB RAM support 100GB workloads by mapping virtual addresses to physical pages, swapping to disk when needed. The MMU translates every memory access — a 99.9% TLB hit rate keeps it fast; a page fault costs ~10ms.
