🌳 Root Servers

13 root server clusters (A-M) operated by independent orgs. Anycast routing — 1,500+ instances worldwide. Know where all TLD nameservers are.

🏛️ TLD Servers

gTLDs (.com, .org) and ccTLDs (.uk, .jp). Each TLD has authoritative nameservers listing where to find each domain's DNS records.

📝 Authoritative NS

Definitive source for a domain's records (A, AAAA, CNAME, MX, TXT). Configured via DNS provider (Route53, Cloudflare DNS, etc.).

💾 DNS Caching

Browser cache (OS-level), resolver cache (ISP/Cloudflare/Google), TTL (seconds to days). Negative caching for NXDOMAIN (SOA minimum).

Record TypesA (IPv4), AAAA (IPv6), CNAME (alias), MX (mail), TXT (SPF/DKIM), NS (nameserver)
TTL60s (frequent changes) to 86400s (24h, stable records). Lower TTL = more queries
DNS over HTTPSEncrypted queries via TLS. Prevents snooping/poisoning. Resolver: 1.1.1.1, 8.8.8.8
AnycastSame IP advertised from multiple locations. Routes to nearest. Used by root/TLD/resolvers.
