🎓 Recommended Learning Path

Foundation Level (Day 1-2)

  1. YouTube Recommendations - Most comprehensive system. Covers all fundamental concepts (retrieval, ranking, re-ranking, online learning)
  2. Google Search Ranking - Different retrieval challenge (inverted index vs FAISS). Introduces learning-to-rank
  3. Ads CTR Prediction - Online learning focus. Real-time feature engineering

Intermediate Level (Day 3-4)

  1. LLM Serving - Modern inference optimization. Batching, KV cache, speculative decoding
  2. TikTok Feed - Reinforcement learning approach. Exploration/exploitation
  3. Netflix Recommendations - Batch approach. Cold-start and diversity

Advanced Level (Day 5-6)

  1. Maps Ranking - Geospatial challenges. Regional distribution
  2. Comparative Analysis - Design unified platform for multiple systems
  3. Production Patterns - Failures, monitoring, scaling, A/B testing
