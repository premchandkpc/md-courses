# PostgreSQL Basics

PostgreSQL is a powerful, open-source object-relational database system known for advanced features and SQL compliance.

## Topics Covered

- **Database & Schema Creation** - Advanced DDL, namespaces
- **Data Types** - Built-in types, arrays, JSON, custom types
- **Tables & Relationships** - Inheritance, constraints, triggers
- **Basic Queries** - SELECT, WHERE, INSERT, UPDATE, DELETE
- **Indexes** - B-tree, Hash, GiST, BRIN indexes
- **Joins** - All join types with advanced syntax

## Key Characteristics

- **Advanced SQL**: Window functions, CTEs, JSON operators
- **Extensibility**: Custom types, functions, operators
- **ACID Transactions**: Full transaction support
- **Open Source**: Community-driven, no licensing costs
- **Performance**: Excellent for complex queries

## Learning Path

1. Master schema design and data types
2. Learn PostgreSQL-specific features (JSON, arrays, ranges)
3. Understand indexing strategies
4. Master joins and subqueries
5. Progress to window functions and CTEs
