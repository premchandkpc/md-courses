Zoom Architecture
Video Conferencing · SFU · WebRTC · E2EE · Global Scale
Zoom serves 300M+ daily meeting participants across 40+ data center regions with P99 media latency < 150ms. Architecture centers on Selective Forwarding Units (SFU) for scalable media distribution, WebSocket-based signaling for real-time control, and E2EE using X25519 ECDH key exchange.
