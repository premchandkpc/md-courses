# MongoDB Basics

MongoDB is a popular document-oriented NoSQL database that stores data in flexible JSON-like documents.

## Topics Covered

- **Collections & Documents** - Flexible schema, document structure
- **BSON Data Types** - ObjectId, arrays, nested documents
- **CRUD Operations** - insertOne, find, updateOne, deleteOne
- **Queries & Filters** - Query operators, comparison, logical operators
- **Indexes** - Single field, compound, text, geospatial indexes
- **Aggregation** - Pipeline stages, grouping, sorting

## Key Characteristics

- **Flexible Schema**: Documents in collection need not have same structure
- **JSON-like**: Easy to work with from application code
- **Scalable**: Built-in sharding for horizontal scaling
- **Rich Queries**: Complex queries, aggregation framework
- **Transactions**: Multi-document ACID transactions (4.0+)

## Learning Path

1. Understand document structure and collections
2. Master CRUD operations
3. Learn query syntax and operators
4. Master indexing for performance
5. Understand aggregation pipeline
