Kernel vs Userspace
Privilege rings · Syscalls · Isolation · Bypass
Why separate kernel and user modes? Hardware protection rings prevent buggy or malicious user code from corrupting the OS. The cost: 1–10µs per syscall for context switches + validation. This guide covers the components, tradeoffs, failure modes, and modern bypass techniques.
