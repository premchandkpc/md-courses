🌍 Edge Locations

Hundreds to thousands of PoPs worldwide. Anycast routing sends user to nearest edge. Cache stores static + dynamic content on SSD/RAM.

🛡️ Origin Shield

Dedicated cache layer between edges and origin. Single point for cache misses — avoids thundering herd when multiple edges miss simultaneously. Reduces origin load by ~80%.

🔄 Cache Invalidation

TTL-based (Cache-Control max-age), Purge by URL/tag (instant), Stale-while-revalidate (serve stale, refresh in background). AWS CloudFront: invalidation paths.

🛡️ DDoS Protection

Edge absorbs attack traffic. Rate limiting, WAF rules, geo-blocking, challenge (JS/captcha). Cloudflare: Magic Transit, AWS: Shield Advanced.

Cache Hit RatioStatic assets: 90-99%. Dynamic API: 20-60%. Higher = lower origin load.
Cache KeysURL + query params + cookies + headers. Customize via cache key policy.
TTL StrategyImmutable assets: 1 year (hash in URL). API: 0-60s. HTML: 0-300s.
Edge ComputeCloudFront Functions (sub-ms), Lambda@Edge (5-30ms), Cloudflare Workers. A/B testing, auth, response transformation.
