# Data Engineer

Master data engineering — from SQL and databases to streaming, data lakes, and MLOps.

## Prerequisites

- Basic programming (Python preferred)
- Familiarity with databases

You can design and build end-to-end data pipelines, from ingestion through transformation to serving. You understand batch and streaming patterns, storage formats, and production data operations.
