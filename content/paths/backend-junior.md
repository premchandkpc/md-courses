# Backend — Junior (0-2 YOE)

Build a strong foundation in operating systems, networking, databases, and API development.

## Prerequisites

None — this path assumes basic programming knowledge.

## Path

1. **Operating Systems Fundamentals**
   - [Linux Kernel Basics](/12-operating-systems/01-linux-kernel-basics.md) — 30m
   - [Process & Threads](/12-operating-systems/02-processes-threads.md) — 30m
   - [Memory Management](/12-operating-systems/03-memory-management.md) — 30m

2. **Networking**
   - [TCP/IP Fundamentals](/11-networking/01-tcp-ip-protocols.md) — 45m
   - [HTTP/1.1 & HTTP/2](/11-networking/02-http-protocols.md) — 30m
   - [DNS & Load Balancing](/11-networking/04-dns-load-balancing.md) — 30m

3. **Databases**
   - [PostgreSQL Architecture](/08-databases/02-postgresql-architecture.md) — 45m
   - [SQL & Query Optimization](/08-databases/03-postgresql-troubleshooting-tuning.md) — 60m
   - [Redis Basics](/08-databases/02-redis-internals.md) — 30m

4. **Backend Development**
   - [Java Concurrency Deep Dive](/03-backend/java/15-concurrency-deep-dive.md) — 45m
   - [Python Async Patterns](/03-backend/python/03-async-patterns.md) — 30m
   - [TypeScript Backend](/03-backend/typescript/01-typescript-backend.md) — 30m

5. **APIs & Integration**
   - [REST & gRPC](/11-networking/05-grpc-api-design.md) — 30m
   - [Message Queues](/10-messaging/01-message-queues.md) — 30m

6. **Version Control & CI/CD**
   - [Git Internals](/25-software-engineering/01-git-internals.md) — 30m
   - [CI/CD Basics](/06-devops/02-ci-cd-pipelines.md) — 30m

7. **Containerization**
   - [Docker Fundamentals](/06-devops/04-docker-fundamentals.md) — 45m
   - [Kubernetes Basics](/07-kubernetes/01-kubernetes-basics.md) — 45m

## Outcome

You can build, deploy, and debug a basic backend service. You understand the request lifecycle from browser to database and back.
